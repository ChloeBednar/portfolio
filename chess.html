5-Axis CNC Chess Set

Project Overview

This project stemmed from my interest in the aesthetics of chess and refining my CNC machining skills.I designed and machined a full chess set and board using a both a 3-axis (Openbuilds CNC) and 5-axis CNC mill (V1 PocketNC). This project allowed me to enhance my skills in CAD modeling, CAM programming, and CNC operation. I overcame many hurdles throughout this project, most notably overcoming the learning curve of the Kickstarter CNC machine in my shop, problem solving machining limitations and tolerances, and creating innovative solutions for piece storage, design, and functionality.
